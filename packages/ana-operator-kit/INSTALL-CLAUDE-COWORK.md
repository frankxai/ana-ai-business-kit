# Use with Claude Cowork

Claude Cowork Projects support a dedicated workspace with its own files, instructions, and memory. Create a private project for this kit; do not use a project that contains unrelated client files.

1. Unzip `ana-operator-kit` into a private folder.
2. In Claude Cowork, create a new Project and attach/select that folder.
3. Copy the contents of `CLAUDE.md` into the Project instructions, then add the privacy boundary from `AGENTS.md` if your Project does not automatically read workspace files.
4. Begin with the safe first prompt below and only add approved source material for one current task.
5. Keep network and connected-service access off unless there is a specific, reviewed need. Review every output before it reaches a client.

## Safe first prompt

> Read the project instructions, `START-HERE.md`, and `reference/PRIVACY-AND-SCOPE.md`. Help me prepare one approved [offer] for [date]. The intended human outcome is [outcome]. Use only the material I provide. Ask up to three questions if needed. Return a labeled draft for my review; do not send, publish, or change any client-facing file.

Cowork’s local project memory is useful for workflow continuity, but it is not a client-record system. Keep sensitive source material to the minimum necessary and remove it from the workspace when it is no longer needed.
