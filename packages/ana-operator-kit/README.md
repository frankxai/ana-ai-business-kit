# Ana Operator Kit — optional facilitation package

This practitioner-only package supports Ana's separate facilitation workflow: offers, sessions, circles, workshops, aftercare, and content prepared from approved material. It is not the primary operating path for the HR agency; use the Ana HR Operations plugin for recurring client, recruiting, document, research, and commercial work. Do not give this folder to a client.

## Decision authority

The practitioner is the author and decision-maker. AI may organize, question, and draft; it may not speak as Ana or decide what a client needs.
