# Session → aftercare workflow

1. Ana decides which notes, if any, are suitable to use.
2. Create a short draft: what was named, one agreed next step, and one optional reflection question.
3. Remove interpretations, diagnosis-like wording, promises, and anything the client did not ask to record.
4. Add a practical boundary or referral note when appropriate.
5. Ana reviews, rewrites in her voice, and sends the final note herself.

Do not turn a private session into public content. A recurring pattern may become content only after it is generalized, de-identified, and approved.
